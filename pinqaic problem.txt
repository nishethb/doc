I am working on a High-Risk Transaction (HRT) authentication POC using PingOne Advanced Identity Cloud (AIC) and PingGateway 2026.3. I need your expert guidance on a specific implementation challenge.

—— BUSINESS REQUIREMENT ——

We are replacing Transmit Security with PingOne AIC + PingGateway for High-Risk Transaction step-up authentication. The flow requires:

1. A transaction identifier (xsmid) is generated by PingGateway for each HRT
2. The user completes step-up authentication via HRTStepUpJourney
3. AIC issues an id_token (JWT) containing xsmid as a plain claim
4. PingGateway validates the JWT via AIC introspect endpoint and matches xsmid from the request with xsmid in the JWT claim
5. Transaction is approved if both match

—— CURRENT FLOW IMPLEMENTED ——

1. DSH (browser app) → login to AIC → establish SSO session
2. DTB → POST PingGateway /api/authenticate → receive xsmid (UUID)
3. DTB → pass xsmid to DSH
4. DSH browser → invoke HRTStepUpJourney via Authorization Code + PKCE:
   GET /oauth2/authorize
     ?authIndexType=service
     &authIndexValue=HRTStepUpJourney
     &login_hint=<xsmid>
     &state=<xsmid>
     &code_challenge=<PKCE>
     &code_challenge_method=S256
5. User completes step-up (password) in journey
6. AIC issues authorization code → browser redirects to callback
7. DSH server exchanges code for id_token
8. id_token passed to DTB
9. DTB → POST PingGateway /api/validate { jwt, xsmid }
10. PingGateway introspects JWT with AIC → validates → matches xsmid → returns APPROVED

—— JOURNEY CONFIGURATION ——

HRTStepUpJourney canvas:

Start
  ↓
Check Session (Scripted Decision)
  Uses: if (typeof existingSession !== 'undefined') → hasSession
  ├── hasSession → Page Node (Password only)
  │                     → Data Store Decision
  │                           True  → StoreXsmid Script → Success
  │                           False → Failure
  └── noSession  → Page Node (Username + Password)
                        → Data Store Decision
                              True  → StoreXsmid Script → Success
                              False → Failure

StoreXsmid Script (Scripted Decision — runs after authentication):

  var outcome = 'Success';
  var xsmid = 'unknown';
  try {
    var hint = requestParameters.get('login_hint');
    if (hint !== null && hint.size() > 0) {
      xsmid = hint.get(0).toString();
    }
    if (xsmid === 'unknown') {
      var st = requestParameters.get('state');
      if (st !== null && st.size() > 0) {
        xsmid = st.get(0).toString();
      }
    }
    sharedState.put('xsmid', xsmid);
    transientState.put('xsmid', xsmid);
    session.setProperty('xsmid', xsmid);
    logger.message('[StoreXsmid] xsmid set: ' + xsmid);
    outcome = 'Success';
  } catch(e) {
    logger.error('[StoreXsmid] Error: ' + e);
    outcome = 'Success';
  }
